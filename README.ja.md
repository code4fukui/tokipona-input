# tokipona-input
The Japanese README is available here: [README.ja.md](README.ja.md)

ブラウザ上で簡単に Toki Pona を入力できるツールです。

## デモ
デモは https://code4fukui.github.io/tokipona-input/ で利用可能です。

## 機能
- トキポナの文字をラテン文字で入力できるテキストエリアを提供します
- 入力内容をトキポナの文字に自動的に変換し、2つ目のテキストエリアに表示します
- 双方向入力 - どちらかのテキストエリアの変更が、もう一方のテキストエリアにも反映されます

## 要件
このプロジェクトはブラウザ上で動作し、外部依存はありません。

## 使用方法
1. ウェブブラウザでデモページを開きます。
2. "in Latin alphabet"のテキストエリアにトキポナのテキストを入力します。
3. "in Tokipona"のテキストエリアが自動的にトキポナの文字に更新されます。
4. "in Tokipona"のテキストエリアに直接入力することもでき、その場合は "in Latin alphabet"のテキストエリアが更新されます。
